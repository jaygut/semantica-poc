## 1. The Global Problem: The Invisible Value of the Ocean

### The Financial Gap
A **$175 billion annual funding gap** exists for ocean conservation (SDG 14). While terrestrial carbon markets have matured, "Blue Carbon" and marine biodiversity markets remain stagnant.

### The Trust Crisis
Investors (banks, insurers, sovereigns) cannot deploy capital because they lack **trust**. Current data is:
1.  **Fragmented:** Locked in PDFs, disparate databases (WoRMS, OBIS), and academic silos.
2.  **Opaque:** "Black box" AI models give answers without proving *how* they reached them.
3.  **Disconnected:** Financial value is rarely physically linked to the underlying ecological health.

**The Result:** Capital sits on the sidelines while critical ecosystems (coral reefs, mangroves, seagrass) degrade. We cannot value what we cannot verify.

## 3. System Architecture: The "Intelligence Factory"

### Phase 1: The Build (Agentic Extraction)
Before the system runs, an army of AI agents builds the knowledge foundation:
1.  **Discovery:** Agents scour 195+ T1/T2 peer-reviewed papers.
2.  **Extraction:** Agents extract **Entities** (Species, Habitats) and **Bridge Axioms**.
3.  **Ingestion:** This "Intelligence Payload" is loaded into a Neo4j Graph Database.

### Phase 2: The Runtime (Deterministic Reasoning)
When an investor asks, *"What is the value of Cabo Pulmo?"*:
1.  **No Hallucination:** The system does *not* guess.
2.  **Graph Traversal:** It walks the graph: `Cabo Pulmo -> Coral Reef -> [Axiom: Flood Protection] -> $29.7M`.
3.  **Provenance:** It returns the value *plus* the exact citation chain (DOI) that justifies it.
